4. Configure Static Files in settings.py
At bottom of settings.py:

from pathlib import Path

STATIC_URL = '/static/'
STATICFILES_DIRS = [
    os.path.join(BASE_DIR, 'static'),

5. Create Static Folder
mkdir static
mkdir static/images

Add an image, e.g. static/images/my_pic.jpg.

6. Create Template
In myapp/templates/myapp/static_example.html:

{% load static %}
<html>
<head>
    <title>This is static template</title>
</head>
<body>
    <h2>Static Image Example</h2>
    <img src="{% static 'images/my_pic.jpg' %}" alt="Image not Found"/>
</body>
</html>

7. Create View in myapp/views.py
from django.shortcuts import render

def static_example(request):
    return render(request, "myapp/static_example.html")

8. App-level urls.py (inside myapp/urls.py)
from django.urls import path
from . import views

urlpatterns = [
    path('static-example/', views.static_example, name="static_example"),
]

9. Project-level urls.py (inside Sample_Project/urls.py)
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),   # ðŸ‘ˆ include app-level urls
]
