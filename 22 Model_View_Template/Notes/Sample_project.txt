Project Name: Sample_Project
Goal: Models → Views → Templates

🚀 Step 1: Create Project & App
Commands:
django-admin startproject Sample_Project
cd Sample_Project
python manage.py startapp myapp

➡ Add myapp to INSTALLED_APPS in Sample_Project/settings.py:
INSTALLED_APPS = [
    ...
    'myapp',
]

🏗 Step 2: Create a Model
File: myapp/models.py
from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    author = models.CharField(max_length=50, default="Anonymous")
    created_at = models.DateTimeField(auto_now_add=True)

    def __str__(self):
        return self.title


➡ Apply migrations:
python manage.py makemigrations
python manage.py migrate

🔑 Step 3: Register Model in Admin (Optional)
File: myapp/admin.py

from django.contrib import admin
from .models import Post
admin.site.register(Post)


➡ Create superuser to access admin:
python manage.py createsuperuser


➡ Run server & login:
👉 http://127.0.0.1:8000/admin

👀 Step 4: Create a View (Model → View)
File: myapp/views.py

from django.shortcuts import render
from .models import Post

# Show all posts
def myapp_view(request):
    posts = Post.objects.all()  # Fetch all posts
    return render(request, 'myapp/myapp_home.html', {'posts': posts})

# Show only posts by author "RK"
def rk_posts_view(request):
    posts = Post.objects.filter(author="RK")
    return render(request, 'myapp/myapp_home.html', {'posts': posts})

🌐 Step 5: Configure URLs
Create myapp/urls.py:

from django.urls import path
from . import views

urlpatterns = [
    path('', views.myapp_view, name='all_posts'),
    path('rk/', views.rk_posts_view, name='rk_posts'),
]


Modify Sample_Project/urls.py to include app URLs:
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]

🎨 Step 6: Create Template (View → Template)
Create folder myapp/templates/myapp/myapp_home.html
File: myapp/templates/myapp/myapp_home.html

<!DOCTYPE html>
<html>
<head>
    <title>My Blog Posts</title>
</head>
<body>
    <h1>Blog Posts</h1>
    {% block content %}
    {% for i in posts %}
         <h3>{{ i.title }}</h3>
         <i>{{ i.created_at }}</i><br>
         <b>By: {{ i.author }}</b>
         <p>{{ i.content }}</p>
         <hr>
    {% empty %}
         <p>No posts available.</p>
    {% endfor %}
    {% endblock content %}
</body>
</html>
