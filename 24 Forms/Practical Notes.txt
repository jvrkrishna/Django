ğŸ”¹ Step 5: Create Contact Form
Create forms.py inside contact app.

contact/forms.py

from django import forms
class ContactForm(forms.Form):
    name = forms.CharField(max_length=50)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

âœ” This form is not connected to database
âœ” Used only to collect user input

ğŸ”¹ Step 6: Create View
Open views.py.
contact/views.py

from django.shortcuts import render
from .forms import ContactForm
def contact_view(request):
    if request.method == "POST":
        form = ContactForm(request.POST)
        if form.is_valid():
            name = form.cleaned_data['name']
            email = form.cleaned_data['email']
            message = form.cleaned_data['message']

            print(name, email, message)  # for testing

            return render(request, 'success.html')
    else:
        form = ContactForm()

    return render(request, 'contact.html', {'form': form})

âœ” Handles GET and POST
âœ” Validates data
âœ” Displays success page
