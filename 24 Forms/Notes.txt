Django Forms
What are Forms in Django?
Forms in Django are used to collect user input, validate data, and save it to the database.

Examples of where forms are used:
Login page
Registration page
Contact form
Feedback form

Django forms help you:
Avoid writing HTML + validation again and again
Protect against invalid or harmful input
Easily connect form data to models (database)

Why Use Django Forms?
Automatic data validation
Built-in security (prevents SQL injection, XSS)
Less code
Easy error handling

2Ô∏è‚É£ Django Form (forms.Form)
Used when:
Data is not directly saved to database
Example: contact form, login form

Example: Simple Contact Form
forms.py
from django import forms
class ContactForm(forms.Form):
    name = forms.CharField(max_length=50)
    email = forms.EmailField()
    message = forms.CharField(widget=forms.Textarea)

views.py
from .forms import ContactForm
def contact_view(request):
    form = ContactForm()
    return render(request, 'contact.html', {'form': form})

‚úî Validation handled automatically
‚úî No database required

3Ô∏è‚É£ Model Form (forms.ModelForm)
Used when:
Data comes from or goes to a database
Best for CRUD operations (Create, Read, Update, Delete)

Example: Student Registration
models.py
from django.db import models
class Student(models.Model):
    name = models.CharField(max_length=50)
    age = models.IntegerField()
    email = models.EmailField()

forms.py
from django import forms
from .models import Student
class StudentForm(forms.ModelForm):
    class Meta:
        model = Student
        fields = '__all__'

‚úî Automatically creates form from model
‚úî Automatically saves data to database
